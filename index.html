Conhecimentos

HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.

CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.

JavaScript

JavaScript é uma linguagem de programação interpretada usada no desenvolvimento web para descrever funcionalidades e comportamentos da página.

Java

Java é uma linguagem compilada e orientada a objetos, usada em sistemas corporativos, apps Android e também no desenvolvimento web, especialmente no backend.

TypeScript

TypeScript é uma linguagem que adiciona tipagem ao JavaScript. É usada principalmente no desenvolvimento web.

PHP

PHP é uma linguagem de programação voltada para o desenvolvimento web. É muito usada na criação de sites dinâmicos e sistemas que rodam no servidor.

C++

C++ é uma linguagem compilada e poderosa, usada em sistemas que exigem alto desempenho. É comum em jogos, softwares embarcados e aplicações que controlam hardware.

SQL

SQL é uma linguagem usada para gerenciar bancos de dados relacionais. É amplamente aplicada em sistemas web, softwares empresariais e análise de dados.

Git | GitHub

Git é um sistema de controle de versões que registra mudanças em arquivos de forma eficiente. GitHub é uma plataforma online que hospeda repositórios Git e facilita colaboração entre desenvolvedores.

Node.js

Node.js é uma plataforma que permite executar JavaScript no servidor. É usada para criar aplicações web rápidas e escaláveis, como APIs e sistemas em tempo real.

Bootstrap

Bootstrap é um framework front-end que facilita a criação de sites responsivos e modernos. É amplamente usado para acelerar o design com componentes prontos e estilos consistentes.

React

React é uma biblioteca JavaScript para construir interfaces de usuário interativas e eficientes. É amplamente usada no desenvolvimento web, especialmente em aplicações dinâmicas de página única (SPA).
